Loan Calculator
----------------------------
A command‑line loan calculator that supports annuity payments and differentiated payments, built as part of the JetBrains Academy / Hyperskill Python Developer track.
The calculator can compute:


  • 	Monthly annuity payment

  
  • 	Number of months needed to repay a loan

  
  • 	Loan principal

  
  • 	Differentiated monthly payments

  
  • 	Overpayment for all loan types

 
The program accepts parameters via command‑line arguments and validates all inputs before performing calculations.
